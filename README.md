Introduction

This README file describes my learning journey of Linked Lists in C++.

What is a Linked List?
-----------------------
A Linked List is a dynamic linear data structure where nodes are connected using pointers.

Each node contains:
- Data
- Address of next node

Topics I Learned
-----------------
1. Basics of Linked List
   - Nodes
   - Pointers
   - Dynamic Memory Allocation

2. Types of Linked Lists
   - Singly Linked List
   - Doubly Linked List
   - Circular Linked List

3. Operations
   - Insertion
   - Deletion
   - Traversal
   - Searching

4. Advanced Concepts
   - Reversing a Linked List
   - Detecting Loops
   - Merging Lists

5. STL Concepts Related to Lists
   - Understanding list containers
